 jar包是客户端,没有环境的情况下,账号admin,密码随便输,进去选单人模式,选择难度即可
 <支持本地联机的雷电>
 实现功能:
 基本功能:敌机爆炸时的音效,子弹音效,背景音乐(都可在设置中关闭) 背景移动 操作方法:上下左右移动空格发射子弹
 普通难度:敌机只会直线移动,子弹发射频率较低.
 地狱难度:敌机可以左右移动,子弹发射频率高,BOSS类子弹频率也更高,血量更多,敌机数量也更多
 账号功能:注册登录注销 排行榜,排行榜有单人和双人两种.
 联机功能:同一局域网内,P1玩家创建房间等待P2玩家加入
 服务端GUI:服务端用Swing的Jtable把数据库的用户信息表给可视化了,同时也支持添加删除用户
